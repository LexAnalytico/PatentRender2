## üö® DO THESE TODAY (Next 2-3 Hours)

### 1. Set Up Google Analytics 4 (30 minutes)
```bash
# Install GA4 package
npm install @next/third-parties
```

Then add to `app/layout.tsx`:
```tsx
import { GoogleAnalytics } from '@next/third-parties/next'

// Add inside <body> tag:
<GoogleAnalytics gaId="G-XXXXXXXXXX" />
```

Get your GA4 ID from: https://analytics.google.com

### 2. Create Google Business Profile (45 minutes)
- Visit: https://business.google.com
- Add your business name, category (Legal Services / IP Attorney)
- Add complete address (critical for local SEO)
- Add business hours, phone number
- Upload logo and photos
- Write description with keywords
- Verify ownership

### 3. Submit Sitemap to Bing (10 minutes)
- Visit: https://www.bing.com/webmasters
- Sign in and add your site
- Submit sitemap: `https://yourdomain.com/sitemap.xml`

### 4. Update Environment Variable (5 minutes)
In Vercel dashboard:
```
NEXT_PUBLIC_SITE_URL=https://yourdomain.com
```
(Replace www.example.com with your actual domain)
